# 理财记录管理系统 (Finance Record App)

一个基于Java的简约美观的理财记录管理应用程序，采用控制台界面设计。

## 功能特性

### 1. 用户管理
- **注册**: 创建新用户账户
- **登录**: 验证用户身份
- **退出登录**: 安全退出当前会话

### 2. 理财产品显示
- 查看所有可购买的理财产品
- 产品信息包括：产品ID、名称、类型、单价、预期收益率、风险等级
- 支持多种产品类型：基金、股票、债券、保险

### 3. 交易记录
- **买入产品**: 购买理财产品并记录交易
- **卖出产品**: 卖出持有的产品并记录交易
- **交易历史**: 查看个人所有交易记录

### 4. 个人信息维护
- 查看个人资料
- 修改姓名、邮箱、电话等信息
- 修改密码（需要验证旧密码）

### 5. 持仓管理
- 查看当前持有的所有产品
- 显示持有份额、平均成本、当前价值、盈亏情况
- 自动计算总资产价值

## 技术架构

```
com.finance.app
├── model/           # 数据模型
│   ├── User.java               # 用户模型
│   ├── FinancialProduct.java   # 理财产品模型
│   ├── Transaction.java        # 交易记录模型
│   └── Holding.java            # 持仓模型
├── repository/      # 数据持久层
│   ├── UserRepository.java     # 用户数据存储
│   ├── ProductRepository.java  # 产品数据存储
│   └── TransactionRepository.java # 交易数据存储
├── service/         # 业务逻辑层
│   └── FinanceService.java     # 核心业务服务
└── ui/              # 用户界面层
    └── MainApp.java            # 主界面和交互逻辑
```

## 数据存储

应用使用JSON文件进行本地数据持久化：
- `users.json` - 用户信息
- `products.json` - 理财产品信息
- `transactions.json` - 交易记录
- `holdings.json` - 用户持仓信息

## 编译和运行

### 环境要求
- Java 17+
- Maven 3.8+

### 编译项目
```bash
cd finance-app
mvn clean package
```

### 运行应用
```bash
java -jar target/finance-app-1.0-SNAPSHOT.jar
```

或者使用Maven运行：
```bash
mvn exec:java -Dexec.mainClass="com.finance.app.ui.MainApp"
```

## 使用说明

### 首次使用
1. 启动应用后选择"注册"创建新账户
2. 输入用户名、密码（至少6位）和姓名
3. 注册成功后使用新账户登录

### 主要操作流程
1. **查看产品**: 浏览所有可投资的理财产品
2. **买入产品**: 输入产品ID和购买份额进行投资
3. **查看持仓**: 查看当前持有的产品及盈亏情况
4. **卖出产品**: 选择持有的产品进行卖出操作
5. **查看交易记录**: 回顾历史交易明细
6. **个人信息**: 更新个人资料或修改密码

### 界面特色
- 采用ANSI颜色代码美化输出
- 清晰的表格化数据显示
- 直观的操作提示和反馈
- 风险等级颜色标识（绿色-低，黄色-中，红色-高）
- 盈亏金额颜色标识（绿色-盈利，红色-亏损）

1. 密码长度必须至少6个字符
2. 卖出产品时不能超出持有份额

4. 产品价格在本次实现中为固定值，实际应用中可接入实时数据
